# 地铁站点周边服务模块开发计划

## 项目概述

基于高德地图 LBS 服务，为广州地铁智慧管理平台增加**地铁站点周边服务模块**，提供地铁站周边 1-3 公里范围内的便民服务查询，包括餐饮、购物、医疗、景点等生活服务设施。

---

## 一、需求分析与功能规划

### 1.1 模块定位

为乘客提供便捷的站点周边生活服务查询功能，帮助用户快速找到地铁站附近的美食、购物、医疗、休闲等设施，并提供步行导航服务。

### 1.2 核心功能

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|-------|
| **站点选择** | 从地铁线路中选择目标站点 | P0 |
| **周边搜索** | 以站点为中心搜索周边 POI | P0 |
| **分类筛选** | 按餐饮/购物/医疗/景点等分类筛选 | P0 |
| **地图展示** | 在地图上展示站点和周边 POI 位置 | P0 |
| **POI 详情** | 显示商家详情、评分、距离、营业时间 | P1 |
| **路径规划** | 从站点到目标 POI 的步行导航 | P1 |

## 五、高德 JSAPI 使用方式

### 5.1 POI 搜索（前端直接调用）

**无需后端 API，直接在 Vue 组件中使用：**

```javascript
// StationServices.vue

// 1. 加载高德地图时引入插件
const initMap = () => {
  return new Promise((resolve, reject) => {
    const script = document.createElement('script')
    script.src = 'https://webapi.amap.com/maps?v=2.0&key=你的Key&plugin=AMap.PlaceSearch,AMap.Walking'
    script.onload = () => {
      createMap()
      initPlaceSearch()  // 初始化POI搜索
      resolve()
    }
    script.onerror = reject
    document.head.appendChild(script)
  })
}

// 2. 创建 PlaceSearch 实例
let placeSearch = null
const initPlaceSearch = () => {
  placeSearch = new AMap.PlaceSearch({
    type: selectedCategoryCode.value,  // POI分类编码，如 '050000'（餐饮）
    radius: searchRadius.value,        // 搜索半径（米）
    pageSize: 20,                      // 每页条数
    pageIndex: 1,                      // 页码
    extensions: 'all'                  // 返回详细信息（包含电话、营业时间等）
  })
}

// 3. 执行周边搜索
const searchNearbyPOI = (station) => {
  const center = [station.longitude, station.latitude]
  
  placeSearch.searchNearBy(
    '',                          // 关键字（空字符串表示按分类搜索）
    center,                      // 中心点坐标 [经度, 纬度]
    searchRadius.value,          // 搜索半径
    (status, result) => {
      if (status === 'complete') {
        poiList.value = result.poiList.pois
        renderPOIMarkers(result.poiList.pois)
      } else {
        console.error('POI搜索失败:', status)
      }
    }
  )
}
```

### 5.2 步行路径规划（前端直接调用）

```javascript
// 步行导航
let walking = null

const initWalking = () => {
  walking = new AMap.Walking({
    map: map.value,        // 地图实例
    panel: 'panel',        // 路线详情面板（可选）
    hideMarkers: false     // 是否隐藏起点终点标记
  })
}

// 规划步行路线
const planWalkingRoute = (start, end) => {
  walking.search(
    [start.longitude, start.latitude],    // 起点
    [end.longitude, end.latitude],        // 终点
    (status, result) => {
      if (status === 'complete') {
        console.log('步行路线:', result.routes[0])
        // 路线已在地图上绘制
      }
    }
  )
}
```

